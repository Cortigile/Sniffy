# Public Images for the User Manual

Store only images in this folder that are allowed to be published in the public PDF manual.

- Reference the images from [../USER_MANUAL_en.md](../USER_MANUAL_en.md) using a relative path, for example `![Application startup](assets/images/startup_screen.png)`
- The manual build intentionally fails if a referenced local image is missing or points outside `sniffy/docs`
